# Xai Intelligence Dashboard 
### High-Fidelity Data Visualization & Adaptive UI

Xai is a premium data intelligence platform designed to showcase the transition from raw, unstructured data (Entropy) to actionable strategic patterns (Order).

## üöÄ Technical Approach
My approach centered on **Performance-First Motion**. Instead of using heavy video assets, the site utilizes:
- **Mathematical Morphing**: Using Three.js `BufferAttributes` to manipulate 3,000+ individual data points at 60fps.
- **Fluid Micro-interactions**: A custom-built Particle Physics engine for the cursor that responds to user velocity and momentum.
- **Scroll-Linked Geometry**: Integrating `framer-motion` with `react-three-fiber` to synchronize DOM text with 3D space.
